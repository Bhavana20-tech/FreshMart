Grocery Product Catalog

This project is a dynamic grocery product catalog that displays fruits, vegetables, and exotic items with detailed information such as price, nutrients, shelf life, and origin. It includes a smart image fallback system that automatically loads category-based images if a product image is missing or fails to load.

Users can click on product images to view them in full size, ensuring a better browsing experience. The project is built using HTML, CSS, and JavaScript and is designed to be responsive, user-friendly, and easy to extend with new products or categories.

Features

Product listing with categories (Fruits, Vegetables, Exotic)

Image fallback handling for broken or missing images

Clickable images that open in a new tab

Clean and responsive UI

Technologies Used

HTML

CSS

JavaScript

How to Use

Clone or download the project.

Open index.html in your browser.

Browse products and click images to view them.
